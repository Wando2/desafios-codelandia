O desafio 15 vai ser um pouco diferente. 
Você terá que personalizar o seu perfil no Github.


Você pode usar meu repositório que tem templates de README para criar o seu.
README template: https://github.com/iuricode/readme-template

Vou deixar também o vídeo da Ballerini como recomendação, ela ensina em vídeo como criar o README para o perfil.
Vídeo da Rafaella Ballerini: https://www.youtube.com/watch?v=TsaLQAetPLU


Lembrando que para você criar um README para o seu perfil o repositório tem que estar em público e com o mesmo nome de usuário do GitHub, assim esse README aparecerá automaticamente na sua página de perfil.
